You can import and export the settings of The Events Calendar.

== Description ==

This Extension will give you the possibility to export the settings
of The Events Calendar in a JSON format, and then import them to a different site.
Note: In this version only The Events Calendar is supported.
Settings for any other Modern Tribe Plugins and premium add-ons are not included.
